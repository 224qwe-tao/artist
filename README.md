# NovelAI 畫師串網站

這是一個離線可用的靜態網站，用於整理 NovelAI 畫師 tag、建立畫師資料方格，以及快速生成畫師串。

## 使用方法

1. 解壓 ZIP。
2. 直接雙擊 `index.html` 開啟。
3. 在頂部「隨機 tag 組合器」設定抽取數量與權重模式。
4. 在「畫師資料方格」用 n×n 控制每頁顯示數量，點擊「加入」建立自己的畫師串。
5. 按畫師卡片上的「編輯」可以加入顯示名稱、備註、相關 URL、圖片 icon。

## 已包含功能

- 736 個畫師 tag 資料，來源為你提供的 `736名画师名称.txt`。
- n×n 方格顯示：例如 n=4 時每頁顯示 4×4 張畫師資料卡。
- 搜尋、排序、翻頁。
- 隨機抽取畫師 tag。
- 普通、加權、降權、混合權重格式。
- 一鍵複製輸出 prompt。
- 已選畫師串管理。
- 新增自訂 `artist:name` tag，保存在瀏覽器 localStorage。
- 編輯畫師方格：可修改顯示名稱、tag、辨認備註。
- 相關 URL 管理：每行一個 URL，自動顯示 Pixiv / X / Fanbox / Patreon / Bluesky 等圖示。
- 畫師圖片 icon：可上傳圖片或填入圖片 URL，方便辨認畫師。
- 下載目前資料 JSON，包含方格編輯、URL、圖片 icon、本機自訂畫師。
- 深色 / 淺色主題切換。

## 圖片與儲存說明

網站不會上傳任何資料。自訂畫師、方格編輯、URL、上傳的圖片 icon 都只保存在目前瀏覽器的 localStorage。上傳圖片會自動壓縮成 360×360 JPEG，以減少卡頓與儲存空間使用。如果儲存失敗，請改用圖片 URL 或較小圖片。

## 資料說明

本網站只整理 tag 與 prompt 組合，不包含圖片生成或爬取圖片。請尊重原創作品與各平台規則。
